<b>Running the Application</b><br />
<br />
Launch the project from Visual Studio.<br />
<br />
All required OpenGL libraries are included as NuGet packages.<br />
<br />
Controls<br />
<br />
Use the W, A, S, and D keys to control the tank:<br />
<br />
<b>W – Move forward</b><br />
<b>S – Move backward</b><br />
<b>A – Rotate left</b><br />
<b>D – Rotate right</b><br />
